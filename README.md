# 📊 Market Databank

A full-stack web platform for uploading, storing, and exploring market datasets (CSV/JSON) with interactive previews and filtering capabilities.

---

## 🚀 Features

* 📤 Upload datasets (.csv, .json)
* 📝 Add title, description, and tags
* 📊 Preview dataset in table format
* 🔍 Search and filter data (by columns)
* 📁 Dataset management dashboard
* 👤 User authentication (Login/Logout)
* 🛡️ Admin-controlled platform (only admin uploads datasets)

---

## 🛠️ Tech Stack

**Frontend:**

* HTML
* CSS
* JavaScript

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB

---

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/market-databank.git
```

2. Navigate to the project:

```
cd market-databank
```

3. Install dependencies:

```
npm install
```

4. Start the server:

```
node server.js
```

5. Open in browser:

```
http://localhost:5000
```
